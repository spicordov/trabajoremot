Usuario : 1

Se creo nuevo repositorio a GitHub llamado Trabajo-en-equipo.
Se añadio varios ficheros a ese repositorio, para ello usamos los siguientes comandos:
git init : Iniciamos git.
git status: Vemos el estado en el que estan los ficheros.
git add +nombre fichero: Seleccionamos lo ficheros a subir a github.
git commit -m:subimos los ficheros a nuestro repositorio.

Con esto tenemos los ficheros subidos a nuestro repositorio,
ahora tenemos que añadir un colaborador, para ellos hacemos lo
siguiente:

En nuestro repositorio en linea creado vamos a Settings, ahi
seleccionamos la opción Manage acces, nos piede contraseña,
una vez dentro seleccionamos la opcion Invite a collaborator,
para ello introducimos su nombre de usuario, y le llagara la invitación al correo con el que esta vinculada la cuenta.

Usuario : 2
Recibo correo con la invitación para participar como colaborador en un repositorio llamado Trabajo-en-equipo en Github
Inicio git en la terminal
git clone y la dirección del repositorio: para clonar el repositorio en mi maquina
git status: Vemos el estado en el que estan los ficheros.
git add: para añadir ficheros
git commit -m: comentamos los cambios que vamos a hacer en el repositorio/ En este caso agregé unos comentarios en los ficheros
git push con la direccion para: subir y guardar los cambios en el repositorio situado en Github.

git touch: creacion del fichero Readme.md
